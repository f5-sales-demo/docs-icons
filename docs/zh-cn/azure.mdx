[Azure 架构图标](https://learn.microsoft.com/en-us/azure/architecture/icons/) 提供 606 个彩色服务图标，用于记录 Azure 基础设施。使用 `@f5-sales-demo/icons-azure` 中的 `Icon` 组件，通过名称渲染任意图标。默认尺寸为 18x18，带渐变填充。

## 使用方法

```astro
---
import Icon from '@f5-sales-demo/icons-azure/Icon.astro';
---

<Icon name="virtual-machine" />
<Icon name="kubernetes-services" size="2rem" />
<Icon name="load-balancers" size="1.5rem" label="Load Balancer" />
```

这些图标使用原始厂商颜色（而非 `currentColor`）。颜色嵌入在 SVG 标记中，在亮色和暗色模式下均保持不变。
