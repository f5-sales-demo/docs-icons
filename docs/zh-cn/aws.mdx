[AWS 架构图标](https://aws.amazon.com/architecture/icons/) 提供 885 个彩色服务图标，用于记录 AWS 基础设施。使用 `@f5xc-salesdemos/icons-aws` 中的 `Icon` 组件，通过名称渲染任意图标。默认尺寸为 48x48，支持按图标单独覆盖尺寸。

## 用法

```astro
---
import Icon from '@f5xc-salesdemos/icons-aws/Icon.astro';
---

<Icon name="lambda" />
<Icon name="ec2" size="2rem" />
<Icon name="vpc" size="1.5rem" label="VPC" />
```

这些图标使用原始厂商颜色（而非 `currentColor`）。颜色嵌入在 SVG 标记中，在浅色和深色模式下均保持不变。
