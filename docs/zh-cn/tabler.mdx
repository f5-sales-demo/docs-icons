[Tabler Icons](https://icon-sets.iconify.design/tabler/) 提供 6,034 个具有统一 2px 笔触宽度的清晰线条图标。使用来自 `@f5-sales-demo/icons-tabler` 的 `Icon` 组件，通过名称渲染任意图标。

## 使用方法

```astro
---
import Icon from '@f5-sales-demo/icons-tabler/Icon.astro';
---

<Icon name="shield" />
<Icon name="cloud" size="2rem" />
<Icon name="server" size="1.5rem" label="Server" />
```

图标以内联 SVG 的形式渲染，使用 `currentColor` — 自动适配亮色和暗色模式。
