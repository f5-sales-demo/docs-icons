三个图标包提供了用于云基础设施文档编写的彩色服务图标。与单色图标包不同，这些图标包保留了原始厂商颜色，适用于架构图。

## 可用图标包

| 图标包 | 图标数量 | 默认尺寸 | 页面 |
|------|-------|-------------|------|
| AWS 架构 | 885 | 48x48 | [AWS](/aws/) |
| Azure 架构 | 606 | 18x18 | [Azure](/azure/) |
| GCP 架构 | 216 | 24x24 | [GCP](/gcp/) |

## 用法

```astro
---
import AwsIcon from '@f5-sales-demo/icons-aws/Icon.astro';
import AzureIcon from '@f5-sales-demo/icons-azure/Icon.astro';
import GcpIcon from '@f5-sales-demo/icons-gcp/Icon.astro';
---

<AwsIcon name="lambda" />
<AzureIcon name="virtual-machine" />
<GcpIcon name="cloud-storage" />
```

## 颜色渲染

这些图标包使用原始厂商颜色（非 `currentColor`）。图标卡片上的 `no-invert` 类可在深色模式下保留颜色。在网格布局之外使用这些图标时，无需特殊处理 — 颜色已嵌入 SVG 标记中。

请参阅各图标包的专属页面，查看按服务类别整理的精选图标网格。

浏览完整图标集：[AWS 图标](https://aws.amazon.com/architecture/icons/) | [Azure 图标](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [GCP 图标](https://cloud.google.com/icons)
