[`@f5-sales-demo/icons-f5xc`](https://www.npmjs.com/package/@f5-sales-demo/icons-f5xc) 包中所有图标的完整可视化参考。这 30 个图标代表 F5 Distributed Cloud 服务，以 Iconify JSON 格式提供，并附带 Astro 组件。

## 用法

```astro
---
import Icon from '@f5-sales-demo/icons-f5xc/Icon.astro';
---

<Icon name="web-app-and-api-protection" size="2em" />
<Icon name="multi-cloud-network-connect" size="1.5em" label="Multi-Cloud Network Connect" />
```

或直接导入 Iconify JSON：

```js
import icons from '@f5-sales-demo/icons-f5xc';
// icons.prefix === 'f5xc'
// icons.icons['bot-defense'].body === '<svg body...>'
```

:::note[深色模式]
这些多色图标使用 CSS 自定义属性（`--color-N600`、`--color-brand`、`--color-blue-light`、`--color-N200`），可自动适配浅色和深色模式。深蓝色和浅蓝色在深色模式下会互换，而 F5 红色强调色保持不变。
:::
