[Phosphor Icons](https://icon-sets.iconify.design/ph/) 提供 9,161 个图标，支持灵活的字重选项 — regular、bold、duotone、fill、light 和 thin。使用 `@f5-sales-demo/icons-phosphor` 中的 `Icon` 组件，通过名称渲染任意图标。

## 用法

```astro
---
import Icon from '@f5-sales-demo/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

图标以内联 SVG 的形式渲染，使用 `currentColor` — 可自动适应亮色和暗色模式。

:::note[字重变体]
Phosphor 图标提供六种字重。regular 字重使用基础名称（例如 `globe`）。其他字重附加后缀：`globe-bold`、`globe-duotone`、`globe-fill`、`globe-light`、`globe-thin`。以下示例展示的是 regular 字重。
:::
