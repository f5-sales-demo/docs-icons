三個圖示套件提供用於記錄雲端基礎架構的彩色服務圖示。與單色圖示套件不同，這些套件保留了原始供應商顏色，適用於架構圖。

## 可用套件

| 套件 | 圖示數量 | 預設尺寸 | 頁面 |
|------|-------|-------------|------|
| AWS 架構 | 885 | 48x48 | [AWS](/aws/) |
| Azure 架構 | 606 | 18x18 | [Azure](/azure/) |
| GCP 架構 | 216 | 24x24 | [GCP](/gcp/) |

## 使用方式

```astro
---
import AwsIcon from '@f5xc-salesdemos/icons-aws/Icon.astro';
import AzureIcon from '@f5xc-salesdemos/icons-azure/Icon.astro';
import GcpIcon from '@f5xc-salesdemos/icons-gcp/Icon.astro';
---

<AwsIcon name="lambda" />
<AzureIcon name="virtual-machine" />
<GcpIcon name="cloud-storage" />
```

## 顏色呈現

這些圖示套件使用原始供應商顏色（非 `currentColor`）。圖示卡片上的 `no-invert` 類別可在深色模式下保留顏色。在網格佈局之外使用這些圖示時，不需要特殊處理 — 顏色已嵌入 SVG 標記中。

請參閱各套件的專屬頁面，查看按服務類別整理的精選圖示網格。
