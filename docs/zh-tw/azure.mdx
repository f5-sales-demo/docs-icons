[Azure 架構圖示](https://learn.microsoft.com/en-us/azure/architecture/icons/) 提供 606 個彩色服務圖示，用於記錄 Azure 基礎架構。使用 `@f5xc-salesdemos/icons-azure` 中的 `Icon` 元件，透過名稱渲染任何圖示。預設大小為 18x18，並帶有漸層填色。

## 使用方式

```astro
---
import Icon from '@f5xc-salesdemos/icons-azure/Icon.astro';
---

<Icon name="virtual-machine" />
<Icon name="kubernetes-services" size="2rem" />
<Icon name="load-balancers" size="1.5rem" label="Load Balancer" />
```

這些圖示使用原始供應商顏色（非 `currentColor`）。顏色內嵌於 SVG 標記中，在淺色和深色模式下均會保留。
