[Phosphor Icons](https://icon-sets.iconify.design/ph/) 提供 9,161 個圖示，具備靈活的粗細選項 — regular、bold、duotone、fill、light 和 thin。使用 `@f5xc-salesdemos/icons-phosphor` 中的 `Icon` 元件，透過名稱渲染任何圖示。

## 使用方式

```astro
---
import Icon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

圖示以內嵌 SVG 方式渲染，使用 `currentColor` — 它們會自動適應淺色和深色模式。

:::note[粗細變體]
Phosphor 圖示有六種粗細。regular 粗細使用基本名稱（例如 `globe`）。其他粗細會附加後綴：`globe-bold`、`globe-duotone`、`globe-fill`、`globe-light`、`globe-thin`。以下範例顯示的是 regular 粗細。
:::
