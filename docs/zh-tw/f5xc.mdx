[`@f5xc-salesdemos/icons-f5xc`](https://www.npmjs.com/package/@f5xc-salesdemos/icons-f5xc) 套件中每個圖示的完整視覺參考。這 30 個圖示代表 F5 Distributed Cloud 服務，以 Iconify JSON 格式提供，並附帶 Astro 元件。

## 使用方式

```astro
---
import Icon from '@f5xc-salesdemos/icons-f5xc/Icon.astro';
---

<Icon name="web-app-and-api-protection" size="2em" />
<Icon name="multi-cloud-network-connect" size="1.5em" label="Multi-Cloud Network Connect" />
```

或直接匯入 Iconify JSON：

```js
import icons from '@f5xc-salesdemos/icons-f5xc';
// icons.prefix === 'f5xc'
// icons.icons['bot-defense'].body === '<svg body...>'
```

:::note[深色模式]
這些多色圖示使用 CSS 自訂屬性（`--color-N600`、`--color-brand`、`--color-blue-light`、`--color-N200`），可自動適應淺色和深色模式。深藍色和淺藍色在深色模式下會互換，而 F5 紅色強調色保持不變。
:::
