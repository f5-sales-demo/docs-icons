[AWS 架構圖示](https://aws.amazon.com/architecture/icons/) 提供 885 個彩色服務圖示，用於記錄 AWS 基礎架構。使用 `@f5-sales-demo/icons-aws` 中的 `Icon` 元件，透過名稱渲染任意圖示。預設尺寸為 48x48，各圖示可個別覆寫尺寸。

## 使用方式

```astro
---
import Icon from '@f5-sales-demo/icons-aws/Icon.astro';
---

<Icon name="lambda" />
<Icon name="ec2" size="2rem" />
<Icon name="vpc" size="1.5rem" label="VPC" />
```

這些圖示使用原廠商顏色（非 `currentColor`）。顏色嵌入於 SVG 標記中，並在淺色與深色模式下均保持不變。
