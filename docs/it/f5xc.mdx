Riferimento visivo completo per ogni icona nel pacchetto [`@f5-sales-demo/icons-f5xc`](https://www.npmjs.com/package/@f5-sales-demo/icons-f5xc). Queste 30 icone rappresentano i servizi F5 Distributed Cloud e sono disponibili come JSON Iconify con un componente Astro.

## Utilizzo

```astro
---
import Icon from '@f5-sales-demo/icons-f5xc/Icon.astro';
---

<Icon name="web-app-and-api-protection" size="2em" />
<Icon name="multi-cloud-network-connect" size="1.5em" label="Multi-Cloud Network Connect" />
```

Oppure importare direttamente il JSON Iconify:

```js
import icons from '@f5-sales-demo/icons-f5xc';
// icons.prefix === 'f5xc'
// icons.icons['bot-defense'].body === '<svg body...>'
```

:::note[Modalità scura]
Queste icone multicolore utilizzano proprietà personalizzate CSS (`--color-N600`, `--color-brand`, `--color-blue-light`, `--color-N200`) che si adattano automaticamente alla modalità chiara e scura. Il blu navy e l'azzurro chiaro si scambiano in modalità scura mentre l'accento rosso F5 rimane costante.
:::
