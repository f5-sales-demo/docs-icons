## วิธีการเรนเดอร์

ชุดไอคอนทั้งหมดใช้คอมโพเนนต์ `Icon.astro` แบบแยกตามแพ็กเกจที่เรนเดอร์ SVG แบบอินไลน์ แต่ละชุดมีการนำเข้าของตัวเอง:

```astro
---
// Starlight built-in
import { Icon } from '@astrojs/starlight/components';

// Other packs — each has its own import
import LucideIcon from '@f5xc-salesdemos/icons-lucide/Icon.astro';
import MdiIcon from '@f5xc-salesdemos/icons-mdi/Icon.astro';
import F5xIcon from '@f5xc-salesdemos/icons-f5xc/Icon.astro';
---

<Icon name="star" />
<LucideIcon name="rocket" />
<MdiIcon name="database" />
<F5xIcon name="bot-defense" />
```

ไอคอนใช้ `currentColor` และปรับตัวตามโหมดสว่างและโหมดมืดโดยอัตโนมัติโดยไม่ต้องจัดการพิเศษใดๆ

## พฤติกรรมโหมดมืด/สว่าง

คอมโพเนนต์ `Icon` ทั้งหมดเรนเดอร์ SVG แบบอินไลน์โดยใช้ `currentColor` ซึ่งปรับตัวตามธีมปัจจุบันโดยอัตโนมัติ — ไม่จำเป็นต้องจัดการ CSS หรือคลาสพิเศษใดๆ

ไอคอน F5 XC service ใช้ CSS custom properties ที่ปรับตัวตามโหมดสว่างและโหมดมืดโดยอัตโนมัติ — ไม่จำเป็นต้องจัดการคลาสพิเศษ สำหรับไอคอนหลายสีอื่นๆ (เช่น ตัวแปรสีของ HashiCorp Flight) คลาส `no-invert` บน `.icon-card-image` จะรักษาสีดั้งเดิมในโหมดมืด

## ควรใช้ชุดไหนดี?

| กรณีการใช้งาน | ชุดที่แนะนำ | เหตุผล |
|----------|------------------|-----|
| UI ทั่วไป (ลูกศร, ปิด, ค้นหา) | Starlight Built-in | ไม่ต้องตั้งค่า พร้อมใช้งานเสมอ |
| แผนภาพผลิตภัณฑ์ F5 | F5 Brand | งานศิลป์เฉพาะด้านเครือข่าย/ความปลอดภัย |
| ไอคอนเส้นขอบทันสมัย | Lucide | สะอาด สม่ำเสมอ มากกว่า 1,600 ไอคอน |
| ไอคอนแบบเติมสี/เส้นขอบจำนวนมาก | Material Design (mdi) | ชุดใหญ่ที่สุดด้วยไอคอน 7,638 รายการ |
| ภาษาการออกแบบ IBM | Carbon | ไอคอนระดับองค์กร 2,582 รายการ |
| ตัวเลือกน้ำหนักเส้นที่ยืดหยุ่น | Phosphor | ไอคอน 9,161 รายการพร้อมตัวแปรน้ำหนัก 6 แบบ |
| ไอคอนเส้นคมชัด | Tabler | ไอคอน 6,034 รายการพร้อมเส้นขอบ 2px สม่ำเสมอ |
| แผนภาพบริการ F5 XC | F5 XC Services | ไอคอนบริการ Distributed Cloud 30 รายการ |
| โลโก้แบรนด์/บริษัท | Simple Icons | ไอคอนแบรนด์มากกว่า 3,200 รายการสำหรับผู้ให้บริการและบริการต่างๆ |
| โลโก้ผู้ให้บริการคลาวด์/โครงสร้างพื้นฐาน | HashiCorp Flight | ไอคอนผู้ให้บริการ AWS, GCP, Azure, K8s |
| แผนภาพสถาปัตยกรรม AWS | AWS Architecture | ไอคอนบริการสีจำนวน 885 รายการสำหรับโครงสร้างพื้นฐาน AWS |
| แผนภาพสถาปัตยกรรม Azure | Azure Architecture | ไอคอนบริการสีจำนวน 606 รายการสำหรับโครงสร้างพื้นฐาน Azure |
| แผนภาพสถาปัตยกรรม GCP | GCP Architecture | ไอคอนบริการสีจำนวน 216 รายการสำหรับโครงสร้างพื้นฐาน Google Cloud |
