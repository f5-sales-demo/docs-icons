ชุดไอคอน Iconify หลายชุดพร้อมใช้งานเป็นแพ็กเกจ npm พร้อมคอมโพเนนต์ `Icon` ของ Astro แต่ละชุดมีการ import เฉพาะของตัวเองและใช้ API แบบ `name` เดียวกัน

## การเปรียบเทียบชุดไอคอน

| ชุดไอคอน | คำนำหน้า | จำนวนไอคอน | สไตล์ | เหมาะสำหรับ | หน้า |
|------|--------|-------|-------|----------|------|
| Material Design | `mdi` | 7,638 | แบบเติมสี + แบบเส้นขอบ | การใช้งานทั่วไป ครอบคลุมกว้างที่สุด | [Material Design](/mdi/) |
| Carbon | `carbon` | 2,582 | แบบเส้นขอบ | การออกแบบสำหรับองค์กร/IBM เส้นสะอาดตา | [Carbon](/carbon/) |
| Phosphor | `ph` | 9,161 | หลายน้ำหนักเส้น | ตัวเลือกน้ำหนักเส้นที่ยืดหยุ่น (บางถึงเติมสี) | [Phosphor](/phosphor/) |
| Tabler | `tabler` | 6,034 | แบบเส้นขีด | ไอคอนเส้นคมชัด เส้นขีดสม่ำเสมอ | [Tabler](/tabler/) |

## การใช้งาน

แต่ละชุดมีคอมโพเนนต์ `Icon.astro` เฉพาะของตัวเอง:

```astro
---
import MdiIcon from '@f5xc-salesdemos/icons-mdi/Icon.astro';
import CarbonIcon from '@f5xc-salesdemos/icons-carbon/Icon.astro';
import PhIcon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
import TablerIcon from '@f5xc-salesdemos/icons-tabler/Icon.astro';
---

<MdiIcon name="database" />
<CarbonIcon name="cloud" />
<PhIcon name="globe" />
<TablerIcon name="shield" />
```

ไอคอนจะแสดงผลเป็น SVG แบบอินไลน์โดยใช้ `currentColor` — ปรับตัวเข้ากับโหมดสว่างและโหมดมืดโดยอัตโนมัติ

## การเลือกชุดไอคอน

- **ครอบคลุมกว้างที่สุด**: Material Design Icons มีจำนวนไอคอนมากที่สุด (7,638) พร้อมรูปแบบเติมสีและเส้นขอบสำหรับแทบทุกกรณีการใช้งาน
- **การออกแบบสำหรับองค์กร**: Carbon เป็นไปตามภาษาการออกแบบของ IBM ด้วยเส้นขอบที่สะอาดตาและสม่ำเสมอ เหมาะสำหรับแอปพลิเคชันธุรกิจ
- **ความยืดหยุ่นของน้ำหนักเส้น**: Phosphor มีรูปแบบน้ำหนักเส้น 6 แบบ (thin, light, regular, bold, fill, duotone) สำหรับการปรับน้ำหนักภาพอย่างละเอียด
- **เส้นขีดสม่ำเสมอ**: Tabler ใช้ความกว้างเส้นขีด 2px สม่ำเสมอทั่วทั้ง 6,034 ไอคอน เพื่อรูปลักษณ์ที่กลมกลืนกัน

ดูหน้าเฉพาะของแต่ละชุดไอคอนสำหรับตารางไอคอนที่คัดสรรมาและลิงก์สำหรับเรียกดู
