[Phosphor Icons](https://icon-sets.iconify.design/ph/) มีไอคอนจำนวน 9,161 รายการพร้อมตัวเลือกน้ำหนักที่หลากหลาย ได้แก่ regular, bold, duotone, fill, light และ thin ใช้ส่วนประกอบ `Icon` จาก `@f5-sales-demo/icons-phosphor` เพื่อแสดงไอคอนใดก็ได้ตามชื่อ

## การใช้งาน

```astro
---
import Icon from '@f5-sales-demo/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

ไอคอนจะแสดงผลเป็น SVG แบบอินไลน์โดยใช้ `currentColor` — ปรับตัวเองโดยอัตโนมัติสำหรับโหมดสว่างและโหมดมืด

:::note[ตัวแปรน้ำหนัก]
ไอคอน Phosphor มีให้เลือกหกน้ำหนัก น้ำหนัก regular ใช้ชื่อพื้นฐาน (เช่น `globe`) น้ำหนักอื่นๆ จะเติมต่อท้ายด้วย suffix ได้แก่: `globe-bold`, `globe-duotone`, `globe-fill`, `globe-light`, `globe-thin` ตัวอย่างด้านล่างแสดงน้ำหนัก regular
:::
