ชุดไอคอนสามชุดให้ไอคอนบริการสีสำหรับการจัดทำเอกสารโครงสร้างพื้นฐานคลาวด์ ต่างจากชุดไอคอนขาวดำ ไอคอนเหล่านี้จะคงสีดั้งเดิมของผู้ให้บริการสำหรับแผนภาพสถาปัตยกรรม

## ชุดไอคอนที่มีให้

| ชุด | ไอคอน | ขนาดเริ่มต้น | หน้า |
|------|-------|-------------|------|
| AWS Architecture | 885 | 48x48 | [AWS](/aws/) |
| Azure Architecture | 606 | 18x18 | [Azure](/azure/) |
| GCP Architecture | 216 | 24x24 | [GCP](/gcp/) |

## การแสดงผลสี

ชุดไอคอนเหล่านี้ใช้สีดั้งเดิมของผู้ให้บริการ (ไม่ใช่ `currentColor`) คลาส `no-invert` บนการ์ดไอคอนจะคงสีไว้ในโหมดมืด เมื่อใช้ไอคอนเหล่านี้นอกเลย์เอาต์แบบกริด ไม่จำเป็นต้องจัดการพิเศษ — สีถูกฝังอยู่ในมาร์กอัป SVG

ดูหน้าเฉพาะของแต่ละชุดไอคอนสำหรับกริดไอคอนที่จัดเรียงตามหมวดหมู่บริการ

เรียกดูชุดไอคอนทั้งหมด: [ไอคอน AWS](https://aws.amazon.com/architecture/icons/) | [ไอคอน Azure](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [ไอคอน GCP](https://cloud.google.com/icons)
