ชุดไอคอนสามชุดให้ไอคอนบริการสีสำหรับการจัดทำเอกสารโครงสร้างพื้นฐานคลาวด์ แตกต่างจากชุดไอคอนขาวดำ ชุดเหล่านี้จะคงสีดั้งเดิมของผู้ให้บริการไว้สำหรับแผนภาพสถาปัตยกรรม

## ชุดที่มีให้ใช้งาน

| ชุด | ไอคอน | ขนาดเริ่มต้น | หน้า |
|------|-------|-------------|------|
| AWS Architecture | 885 | 48x48 | [AWS](/aws/) |
| Azure Architecture | 606 | 18x18 | [Azure](/azure/) |
| GCP Architecture | 216 | 24x24 | [GCP](/gcp/) |

## การแสดงผลสี

ชุดไอคอนเหล่านี้ใช้สีดั้งเดิมของผู้ให้บริการ (ไม่ใช่ `currentColor`) คลาส `no-invert` บนการ์ดไอคอนจะคงสีไว้ในโหมดมืด เมื่อใช้ไอคอนเหล่านี้นอกเลย์เอาต์แบบกริด ไม่จำเป็นต้องจัดการเป็นพิเศษ — สีจะถูกฝังไว้ในมาร์กอัป SVG

ดูหน้าเฉพาะของแต่ละชุดสำหรับกริดไอคอนที่คัดสรรแล้วจัดเรียงตามหมวดหมู่บริการ

เรียกดูชุดไอคอนทั้งหมด: [AWS Icons](https://aws.amazon.com/architecture/icons/) | [Azure Icons](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [GCP Icons](https://cloud.google.com/icons)
