[Phosphor Icons](https://icon-sets.iconify.design/ph/) bietet 9.161 Icons mit flexiblen Gewichtsoptionen — regular, bold, duotone, fill, light und thin. Verwenden Sie die `Icon`-Komponente aus `@f5xc-salesdemos/icons-phosphor`, um ein beliebiges Icon anhand seines Namens darzustellen.

## Verwendung

```astro
---
import Icon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

Icons werden als inline SVGs mit `currentColor` gerendert — sie passen sich automatisch an den Hell- und Dunkelmodus an.

:::note[Gewichtsvarianten]
Phosphor Icons sind in sechs Gewichtsstufen erhältlich. Das reguläre Gewicht verwendet den Basisnamen (z. B. `globe`). Andere Gewichtsstufen erhalten ein Suffix: `globe-bold`, `globe-duotone`, `globe-fill`, `globe-light`, `globe-thin`. Die folgenden Beispiele zeigen das reguläre Gewicht.
:::
