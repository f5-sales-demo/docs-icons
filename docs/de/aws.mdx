[AWS Architecture Icons](https://aws.amazon.com/architecture/icons/) bietet 885 farbige Service-Icons zur Dokumentation von AWS-Infrastruktur. Verwenden Sie die `Icon`-Komponente aus `@f5xc-salesdemos/icons-aws`, um ein beliebiges Icon nach Name zu rendern. Die Standardgröße beträgt 48x48 mit icon-spezifischen Größenüberschreibungen.

## Verwendung

```astro
---
import Icon from '@f5xc-salesdemos/icons-aws/Icon.astro';
---

<Icon name="lambda" />
<Icon name="ec2" size="2rem" />
<Icon name="vpc" size="1.5rem" label="VPC" />
```

Diese Icons verwenden die originalen Herstellerfarben (nicht `currentColor`). Die Farben sind im SVG-Markup eingebettet und werden sowohl im hellen als auch im dunklen Modus beibehalten.
