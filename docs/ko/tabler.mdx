[Tabler Icons](https://icon-sets.iconify.design/tabler/)는 일관된 2px 선 두께를 가진 6,034개의 선명한 라인 아이콘을 제공합니다. `@f5-sales-demo/icons-tabler`의 `Icon` 구성 요소를 사용하여 이름으로 모든 아이콘을 렌더링하세요.

## 사용법

```astro
---
import Icon from '@f5-sales-demo/icons-tabler/Icon.astro';
---

<Icon name="shield" />
<Icon name="cloud" size="2rem" />
<Icon name="server" size="1.5rem" label="Server" />
```

아이콘은 `currentColor`를 사용하는 인라인 SVG로 렌더링되어 라이트 모드와 다크 모드에 자동으로 적응합니다.
