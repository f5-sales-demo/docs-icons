세 가지 아이콘 팩은 클라우드 인프라 문서화를 위한 컬러 서비스 아이콘을 제공합니다. 단색 아이콘 팩과 달리, 이 팩들은 아키텍처 다이어그램에서 벤더 고유의 원본 색상을 유지합니다.

## 사용 가능한 팩

| 팩 | 아이콘 수 | 기본 크기 | 페이지 |
|------|-------|-------------|------|
| AWS Architecture | 885 | 48x48 | [AWS](/aws/) |
| Azure Architecture | 606 | 18x18 | [Azure](/azure/) |
| GCP Architecture | 216 | 24x24 | [GCP](/gcp/) |

## 색상 렌더링

이 아이콘 팩들은 `currentColor`가 아닌 벤더 고유의 원본 색상을 사용합니다. 아이콘 카드의 `no-invert` 클래스는 다크 모드에서 색상을 보존합니다. 그리드 레이아웃 외부에서 이 아이콘을 사용할 때는 별도의 처리가 필요하지 않습니다 — 색상이 SVG 마크업에 내장되어 있기 때문입니다.

서비스 카테고리별로 정리된 아이콘 그리드는 각 팩의 전용 페이지를 참조하세요.
