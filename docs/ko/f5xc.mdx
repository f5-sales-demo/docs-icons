[`@f5-sales-demo/icons-f5xc`](https://www.npmjs.com/package/@f5-sales-demo/icons-f5xc) 패키지의 모든 아이콘에 대한 완전한 시각적 참조입니다. 이 30개의 아이콘은 F5 Distributed Cloud 서비스를 나타내며, Astro 컴포넌트와 함께 Iconify JSON으로 사용할 수 있습니다.

## 사용법

```astro
---
import Icon from '@f5-sales-demo/icons-f5xc/Icon.astro';
---

<Icon name="web-app-and-api-protection" size="2em" />
<Icon name="multi-cloud-network-connect" size="1.5em" label="Multi-Cloud Network Connect" />
```

또는 Iconify JSON을 직접 가져올 수 있습니다:

```js
import icons from '@f5-sales-demo/icons-f5xc';
// icons.prefix === 'f5xc'
// icons.icons['bot-defense'].body === '<svg body...>'
```

:::note[다크 모드]
이 다중 색상 아이콘들은 라이트 모드와 다크 모드에 자동으로 적응하는 CSS 커스텀 속성(`--color-N600`, `--color-brand`, `--color-blue-light`, `--color-N200`)을 사용합니다. 다크 모드에서는 네이비와 라이트 블루가 전환되며, F5 레드 액센트는 일정하게 유지됩니다.
:::
