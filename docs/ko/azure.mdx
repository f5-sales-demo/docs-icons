[Azure 아키텍처 아이콘](https://learn.microsoft.com/en-us/azure/architecture/icons/)은 Azure 인프라 문서화를 위한 606개의 컬러 서비스 아이콘을 제공합니다. `@f5-sales-demo/icons-azure`의 `Icon` 컴포넌트를 사용하여 이름으로 아이콘을 렌더링합니다. 기본 크기는 그라디언트 채우기가 적용된 18x18입니다.

## 사용법

```astro
---
import Icon from '@f5-sales-demo/icons-azure/Icon.astro';
---

<Icon name="virtual-machine" />
<Icon name="kubernetes-services" size="2rem" />
<Icon name="load-balancers" size="1.5rem" label="Load Balancer" />
```

이 아이콘은 `currentColor`가 아닌 원래 벤더 색상을 사용합니다. 색상은 SVG 마크업에 포함되어 있으며 라이트 모드와 다크 모드 모두에서 유지됩니다.
