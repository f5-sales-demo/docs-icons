[Phosphor Icons](https://icon-sets.iconify.design/ph/)는 regular, bold, duotone, fill, light, thin의 유연한 굵기 옵션을 갖춘 9,161개의 아이콘을 제공합니다. `@f5xc-salesdemos/icons-phosphor`의 `Icon` 컴포넌트를 사용하여 이름으로 원하는 아이콘을 렌더링할 수 있습니다.

## 사용법

```astro
---
import Icon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

아이콘은 `currentColor`를 사용하는 인라인 SVG로 렌더링되어 라이트 모드와 다크 모드에 자동으로 적응합니다.

:::note[굵기 변형]
Phosphor 아이콘은 6가지 굵기로 제공됩니다. regular 굵기는 기본 이름을 사용합니다 (예: `globe`). 다른 굵기는 접미사를 추가합니다: `globe-bold`, `globe-duotone`, `globe-fill`, `globe-light`, `globe-thin`. 아래 예시는 regular 굵기를 보여줍니다.
:::
