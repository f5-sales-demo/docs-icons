[Phosphor Icons](https://icon-sets.iconify.design/ph/)는 regular, bold, duotone, fill, light, thin 등 유연한 두께 옵션을 갖춘 9,161개의 아이콘을 제공합니다. `@f5-sales-demo/icons-phosphor`의 `Icon` 구성 요소를 사용하여 이름으로 아이콘을 렌더링하세요.

## 사용법

```astro
---
import Icon from '@f5-sales-demo/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

아이콘은 `currentColor`를 사용하여 인라인 SVG로 렌더링되며, 라이트 모드와 다크 모드에 자동으로 적응합니다.

:::note[두께 변형]
Phosphor 아이콘은 여섯 가지 두께로 제공됩니다. 기본 두께는 기본 이름(예: `globe`)을 사용합니다. 다른 두께는 접미사를 추가합니다: `globe-bold`, `globe-duotone`, `globe-fill`, `globe-light`, `globe-thin`. 아래 예시는 기본 두께를 보여줍니다.
:::
