[Phosphor Icons](https://icon-sets.iconify.design/ph/) fornece 9.161 ícones com opções flexíveis de peso — regular, bold, duotone, fill, light e thin. Use o componente `Icon` de `@f5xc-salesdemos/icons-phosphor` para renderizar qualquer ícone pelo nome.

## Uso

```astro
---
import Icon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

Os ícones são renderizados como SVGs inline usando `currentColor` — eles se adaptam automaticamente aos modos claro e escuro.

:::note[Variantes de Peso]
Os ícones Phosphor possuem seis pesos. O peso regular usa o nome base (ex.: `globe`). Os outros pesos adicionam um sufixo: `globe-bold`, `globe-duotone`, `globe-fill`, `globe-light`, `globe-thin`. Os exemplos abaixo mostram o peso regular.
:::
