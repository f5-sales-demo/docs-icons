[Material Design Icons](https://icon-sets.iconify.design/mdi/) fornece 7.638 ícones com variantes preenchidas e delineadas, cobrindo praticamente todos os casos de uso. Use o componente `Icon` de `@f5-sales-demo/icons-mdi` para renderizar qualquer ícone pelo nome.

## Uso

```astro
---
import Icon from '@f5-sales-demo/icons-mdi/Icon.astro';
---

<Icon name="database" />
<Icon name="shield-lock" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

Os ícones são renderizados como SVGs inline usando `currentColor` — eles se adaptam automaticamente aos modos claro e escuro.
