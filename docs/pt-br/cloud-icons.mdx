Três pacotes de ícones fornecem ícones coloridos de serviços para documentação de infraestrutura em nuvem. Diferentemente dos pacotes de ícones monocromáticos, estes preservam as cores originais dos fornecedores para diagramas de arquitetura.

## Pacotes Disponíveis

| Pacote | Ícones | Tamanho Padrão | Página |
|--------|--------|----------------|--------|
| AWS Architecture | 885 | 48x48 | [AWS](/aws/) |
| Azure Architecture | 606 | 18x18 | [Azure](/azure/) |
| GCP Architecture | 216 | 24x24 | [GCP](/gcp/) |

```astro
<AwsIcon name="lambda" />

<GcpIcon name="cloud-storage" />
```

## Renderização de Cores

Esses pacotes de ícones utilizam as cores originais dos fornecedores (não `currentColor`). A classe `no-invert` nos cartões de ícones preserva as cores no modo escuro. Ao usar esses ícones fora do layout de grade, nenhum tratamento especial é necessário — as cores estão incorporadas na marcação SVG.

Consulte a página dedicada de cada pacote para grades de ícones organizadas por categoria de serviço.

Navegue pelos conjuntos completos de ícones: [Ícones AWS](https://aws.amazon.com/architecture/icons/) | [Ícones Azure](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [Ícones GCP](https://cloud.google.com/icons)
