Referencia visual completa de cada icono del paquete [`@f5xc-salesdemos/icons-f5xc`](https://www.npmjs.com/package/@f5xc-salesdemos/icons-f5xc). Estos 30 iconos representan servicios de F5 Distributed Cloud y están disponibles como JSON de Iconify con un componente Astro.

## Uso

```astro
---
import Icon from '@f5xc-salesdemos/icons-f5xc/Icon.astro';
---

<Icon name="web-app-and-api-protection" size="2em" />
<Icon name="multi-cloud-network-connect" size="1.5em" label="Multi-Cloud Network Connect" />
```

O importe el JSON de Iconify directamente:

```js
import icons from '@f5xc-salesdemos/icons-f5xc';
// icons.prefix === 'f5xc'
// icons.icons['bot-defense'].body === '<svg body...>'
```

:::note[Modo oscuro]
Estos iconos multicolor utilizan propiedades personalizadas de CSS (`--color-N600`, `--color-brand`, `--color-blue-light`, `--color-N200`) que se adaptan automáticamente al modo claro y oscuro. Los colores azul marino y azul claro se intercambian en el modo oscuro, mientras que el acento rojo de F5 permanece constante.
:::
