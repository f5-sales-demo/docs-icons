[Azure Architecture Icons](https://learn.microsoft.com/en-us/azure/architecture/icons/) fournit 606 icônes de services en couleur pour documenter l'infrastructure Azure. Utilisez le composant `Icon` de `@f5-sales-demo/icons-azure` pour afficher n'importe quelle icône par son nom. La taille par défaut est 18x18 avec des remplissages en dégradé.

## Utilisation

```astro
---
import Icon from '@f5-sales-demo/icons-azure/Icon.astro';
---

<Icon name="virtual-machine" />
<Icon name="kubernetes-services" size="2rem" />
<Icon name="load-balancers" size="1.5rem" label="Load Balancer" />
```

Ces icônes utilisent les couleurs d'origine du fournisseur (et non `currentColor`). Les couleurs sont intégrées dans le balisage SVG et préservées en mode clair comme en mode sombre.
