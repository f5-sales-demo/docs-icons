[Tabler Icons](https://icon-sets.iconify.design/tabler/) は、一貫した 2px ストローク幅を持つ 6,034 個のシャープなラインアイコンを提供します。`@f5xc-salesdemos/icons-tabler` の `Icon` コンポーネントを使用して、名前でアイコンをレンダリングします。

## 使い方

```astro
---
import Icon from '@f5xc-salesdemos/icons-tabler/Icon.astro';
---

<Icon name="shield" />
<Icon name="cloud" size="2rem" />
<Icon name="server" size="1.5rem" label="Server" />
```

アイコンは `currentColor` を使用してインライン SVG としてレンダリングされ、ライトモードとダークモードに自動的に対応します。
