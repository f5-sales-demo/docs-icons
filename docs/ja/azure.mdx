[Azure アーキテクチャアイコン](https://learn.microsoft.com/en-us/azure/architecture/icons/)は、Azure インフラストラクチャのドキュメント作成用に606種類のカラーサービスアイコンを提供しています。`@f5xc-salesdemos/icons-azure` の `Icon` コンポーネントを使用して、名前を指定して任意のアイコンをレンダリングできます。デフォルトサイズは18x18で、グラデーション塗りが適用されます。

## 使い方

```astro
---
import Icon from '@f5xc-salesdemos/icons-azure/Icon.astro';
---

<Icon name="virtual-machine" />
<Icon name="kubernetes-services" size="2rem" />
<Icon name="load-balancers" size="1.5rem" label="Load Balancer" />
```

これらのアイコンはベンダーオリジナルの色を使用しています（`currentColor` ではありません）。色は SVG マークアップに埋め込まれており、ライトモードとダークモードの両方で保持されます。
