[Material Design Icons](https://icon-sets.iconify.design/mdi/) は、ほぼあらゆるユースケースをカバーするフィルドおよびアウトラインバリアントを持つ 7,638 個のアイコンを提供します。`@f5xc-salesdemos/icons-mdi` の `Icon` コンポーネントを使用して、名前でアイコンをレンダリングできます。

## 使い方

```astro
---
import Icon from '@f5xc-salesdemos/icons-mdi/Icon.astro';
---

<Icon name="database" />
<Icon name="shield-lock" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

アイコンは `currentColor` を使用してインライン SVG としてレンダリングされ、ライトモードとダークモードに自動的に対応します。
