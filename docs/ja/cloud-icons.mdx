3つのアイコンパックが、クラウドインフラストラクチャのドキュメント作成用にカラーサービスアイコンを提供しています。モノクロアイコンパックとは異なり、これらはアーキテクチャ図用にベンダーのオリジナルカラーを保持します。

## 利用可能なパック

| パック | アイコン数 | デフォルトサイズ | ページ |
|------|-------|-------------|------|
| AWS Architecture | 885 | 48x48 | [AWS](/aws/) |
| Azure Architecture | 606 | 18x18 | [Azure](/azure/) |
| GCP Architecture | 216 | 24x24 | [GCP](/gcp/) |

## カラーレンダリング

これらのアイコンパックはベンダーのオリジナルカラーを使用しています（`currentColor` ではありません）。アイコンカードの `no-invert` クラスにより、ダークモードでもカラーが保持されます。これらのアイコンをグリッドレイアウトの外で使用する場合、特別な対応は不要です — カラーは SVG マークアップに埋め込まれています。

サービスカテゴリ別に整理されたアイコングリッドについては、各パックの専用ページをご覧ください。

フルアイコンセットを閲覧: [AWS Icons](https://aws.amazon.com/architecture/icons/) | [Azure Icons](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [GCP Icons](https://cloud.google.com/icons)
