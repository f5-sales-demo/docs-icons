3つのアイコンパックは、クラウドインフラストラクチャのドキュメント作成用にカラーサービスアイコンを提供します。モノクロアイコンパックとは異なり、これらはアーキテクチャ図用にベンダーのオリジナルカラーを保持します。

## 利用可能なパック

| パック | アイコン数 | デフォルトサイズ | ページ |
|------|-------|-------------|------|
| AWS アーキテクチャ | 885 | 48x48 | [AWS](/aws/) |
| Azure アーキテクチャ | 606 | 18x18 | [Azure](/azure/) |
| GCP アーキテクチャ | 216 | 24x24 | [GCP](/gcp/) |

import AwsIcon from '@f5-sales-demo/icons-aws/Icon.astro';

<AwsIcon name="lambda" />

## カラーレンダリング

これらのアイコンパックはベンダーのオリジナルカラーを使用します（`currentColor` ではありません）。アイコンカードの `no-invert` クラスにより、ダークモードでもカラーが保持されます。これらのアイコンをグリッドレイアウト以外で使用する場合、特別な処理は不要です — カラーは SVG マークアップに埋め込まれています。

サービスカテゴリ別に整理されたアイコングリッドについては、各パックの専用ページをご覧ください。

完全なアイコンセットを閲覧: [AWS アイコン](https://aws.amazon.com/architecture/icons/) | [Azure アイコン](https://learn.microsoft.com/en-us/azure/architecture/icons/) | [GCP アイコン](https://cloud.google.com/icons)
