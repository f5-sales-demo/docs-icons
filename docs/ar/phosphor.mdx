توفر [أيقونات Phosphor](https://icon-sets.iconify.design/ph/) عدد 9,161 أيقونة مع خيارات أوزان مرنة — عادي، عريض، ثنائي اللون، مملوء، خفيف، ورفيع. استخدم مكوّن `Icon` من `@f5xc-salesdemos/icons-phosphor` لعرض أي أيقونة بالاسم.

## الاستخدام

```astro
---
import Icon from '@f5xc-salesdemos/icons-phosphor/Icon.astro';
---

<Icon name="globe" />
<Icon name="shield" size="2rem" />
<Icon name="cloud" size="1.5rem" label="Cloud" />
```

تُعرض الأيقونات كعناصر SVG مضمّنة باستخدام `currentColor` — فتتكيف تلقائياً مع الوضع الفاتح والداكن.

:::note[متغيرات الأوزان]
تأتي أيقونات Phosphor بستة أوزان. يستخدم الوزن العادي الاسم الأساسي (مثل `globe`). أما الأوزان الأخرى فتُضاف إليها لاحقة: `globe-bold`، `globe-duotone`، `globe-fill`، `globe-light`، `globe-thin`. تعرض الأمثلة أدناه الوزن العادي.
:::
