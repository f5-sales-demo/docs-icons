## طريقة العرض

تستخدم جميع حزم الأيقونات مكوّنات `Icon.astro` الخاصة بكل حزمة التي تعرض رسومات SVG مضمّنة. لكل حزمة استيراد خاص بها:

```astro
---
// Starlight المدمجة
import { Icon } from '@astrojs/starlight/components';

// الحزم الأخرى — لكل منها استيراد خاص
import LucideIcon from '@f5xc-salesdemos/icons-lucide/Icon.astro';
import MdiIcon from '@f5xc-salesdemos/icons-mdi/Icon.astro';
import F5xIcon from '@f5xc-salesdemos/icons-f5xc/Icon.astro';
---

<Icon name="star" />
<LucideIcon name="rocket" />
<MdiIcon name="database" />
<F5xIcon name="bot-defense" />
```

تستخدم الأيقونات `currentColor` وتتكيف تلقائيًا مع الوضع الفاتح والداكن دون الحاجة إلى أي معالجة خاصة.

## سلوك الوضع الداكن/الفاتح

تعرض جميع مكوّنات `Icon` رسومات SVG مضمّنة باستخدام `currentColor`. تتكيف تلقائيًا مع القالب الحالي — دون الحاجة إلى أي معالجة خاصة لـ CSS أو الفئات.

تستخدم أيقونات خدمات F5 XC خصائص CSS المخصصة التي تتكيف تلقائيًا مع الوضع الفاتح والداكن — دون الحاجة إلى أي معالجة خاصة للفئات. بالنسبة للأيقونات متعددة الألوان الأخرى (مثل متغيرات ألوان HashiCorp Flight)، تحافظ فئة `no-invert` على `.icon-card-image` على الألوان الأصلية في الوضع الداكن.
